## 🤖 AI Assistant Guidelines (AI 어시스턴트 필수 수칙)

1. **`CropoutSampleProject/GDDRequest.md` 상시 참조**
   - 게임의 핵심 기획, 레벨 구조, 전반적인 목표가 담겨있는 코어 기획서입니다. 작업 방향성을 파악할 때 항상 가장 먼저 확인하세요.

2. **`CropoutSampleProject/DevelopmentProgress.md` 진행 상황 확인**

3. **사용자 명령 전까지 진행 상황 업데이트 금지**
   - `DevelopmentProgress.md` 파일은 사용자가 명시적으로 업데이트를 지시(명령)할 때만 수정하세요. 임의로 작업 내역을 추가하거나 문서를 변경해서는 안 됩니다.

4. **기존 기획서 보존**
   - 어떠한 경우에도 원본 `GDDRequest.md`의 내용을 임의로 수정하거나 훼손하지 마세요.
